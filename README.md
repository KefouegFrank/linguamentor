# LinguaMentor

Monorepo for LinguaMentor AI-driven language learning platform.

Folders:
- /backend      - Node.js + TypeScript services (API, auth, admin)
- /ai-service   - Python FastAPI-based AI microservices
- /frontend     - Next.js React app (web client)
- /infra        - Terraform + Helm + k8s manifests + CI deploy

See each folder README for service-specific setup.
